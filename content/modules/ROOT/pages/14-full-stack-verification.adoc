== Gateway & Security Checklist

# Gateways are programmed
oc get gateway -n {user_name}-neuralbank -o wide

# HTTPRoutes are accepted
oc get httproute -n {user_name}-neuralbank -o wide

# AuthPolicies are enforced

# RateLimitPolicies are enforced

== API Testing Checklist

=== Backend API (API Key)

[.console-input]
[source,bash]
----
API_KEY=$(oc get secret {user_name}-neuralbank-backend-apikey \
  -n {user_name}-neuralbank \
  -o jsonpath='{.data.api_key}' | base64 -d)

# Should return 200 with customer data
curl -s -w "\nHTTP: %{http_code}\n" \
  -H "X-API-Key: $API_KEY" \
  "https://neuralbank-backend-{user_name}-neuralbank.{cluster_domain}/api/customers" \
  | tail -1

# Should return 401 without key
curl -s -o /dev/null -w "HTTP: %{http_code}\n" \
  "https://neuralbank-backend-{user_name}-neuralbank.{cluster_domain}/api/customers"
----

=== Pre-deployed Stack (OIDC)

[.console-input]
[source,bash]
----
TOKEN=$(curl -s -X POST \
  "https://rhbk.{cluster_domain}/realms/neuralbank/protocol/openid-connect/token" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=password" \
  -d "client_id=neuralbank-frontend" \
  -d "username={user_name}" \
  -d "password=Welcome123!" \
  | python3 -c "import json,sys; print(json.load(sys.stdin)['access_token'])")

# Should return 200
curl -s -w "\nHTTP: %{http_code}\n" \
  -H "Authorization: Bearer $TOKEN" \
  "https://neuralbank.{cluster_domain}/api/v1/customers" \
  | tail -1
----
