= From Zero To Hero with Red Hat Developer Hub

== Welcome to the Workshop

In this workshop you will learn to use **Red Hat Developer Hub** as a self-service development portal to build, deploy, and manage applications on OpenShift.

=== What You Will Learn

* Understand the value proposition of Red Hat Developer Hub
* Explore the architecture: Developer Hub, ArgoCD, Tekton, DevSpaces, Gitea, Keycloak
* Create backend and frontend services using **Software Templates**
* Explore automated pipelines, topology view, and application details
* Deploy and configure API Gateways with security (OIDCPolicy, RateLimitPolicy)
* Update source code using **Red Hat OpenShift Dev Spaces** with automated CI/CD

=== The Business Case: Neuralbank

Neuralbank is a financial institution that needs to modernize its technology stack. As a developer, you will build three components:

1. **Customer Service MCP** - An MCP (Model Context Protocol) server for customer support
2. **Neuralbank Backend** - REST API for credit management
3. **Neuralbank Frontend** - Web interface for credit visualization
