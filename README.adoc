= IA Development From Zero To Hero

A workshop for building MCP Agents on Red Hat OpenShift AI, based on the https://github.com/maximilianoPizarro/rhoai-roadshow-from-zero-to-hero[rhoai-roadshow-from-zero-to-hero] content. Built with https://antora.org[Antora] and antora-viewer format.

== About This Workshop

This workshop takes you through a complete journey from understanding a business use case (Neuralbank) to deploying a production-ready AI solution using Model Context Protocol (MCP) Agents.

* *Business Case*: Neuralbank - a financial institution automating credit risk assessment
* *Your Role*: Kevin, Java developer building the customer-service-mcp MCP Agent
* *Technologies*: OpenShift AI, LlamaStack, Quarkus, Keycloak, Connectivity Link, OpenTelemetry

== Quick Start

. Clone this repository
. Preview locally with antora-viewer:
+
[source,sh]
----
podman run --rm --name antora -v $PWD:/antora -p 8080:8080 -i -t ghcr.io/juliaaano/antora-viewer
----
+
On SELinux systems, append `:z` to the volume mount.

. Edit content in `content/modules/ROOT/pages/`
. Set your cluster domain in the UI (if using showroom deployment)

== Documentation Structure

* *Getting Connected*: Workshop overview, Neuralbank business case, environment configuration
* *MCP Agents & Tools*: Golden Path, DevSpaces, Keycloak, Build MCP Agent, Connectivity Link, MCP Inspector, Deploy & Integrate, OpenTelemetry
* *Resources*: Source code, installation methods (Ansible, Helm, GitOps)
